# K리그 경기응원 서비스

## 기술스택

- Python(Flask)
- mongoDB
- JWT
- bootstrap

## UX/UI

### 1. 메인페이지
- 경기데이터 월별 조회

### 2. 경기정보 페이지
- 해당 경기 정보 및 게시글(댓글) 조회

### 3. 관리자 페이지
- 경기 일정 등록페이지
- 게시글 관리 페이지
- 회원 관리 페이지


## 주요기능

### 1. 회원
- JWT 회원가입(정규식 유효성 검사)
- 로그인
- 게시글 작성 및 삭제

### 2. 관리자
- 회원 활성/비활성화
- 전체 조회 및 삭제
- 게시글 전체 조회 및 삭제
- 경기정보 등록 및 삭제
